## 1. SERVICES PROVIDED

Service Provider agrees to build and deliver the following:

### Base Package Includes:
- **Custom landing page** tailored to Client's business, fully responsive for mobile and web
- **Quote request form** with the following fields:
  - Customer name, phone, email (optional), property address
  - Property type, service needed, project details
  - Best way to reach, referral source
- **Instant email notification** to Client when a lead is submitted
- **Private admin dashboard** accessible via password from phone or computer with:
  - Lead tracking by status (New, Contacted, Quoted, Won, Lost)
  - Search and filter capabilities
  - Follow-up reminders and internal notes
  - CSV export functionality
- **Hosting** on Render.com (free tier)
- **Initial setup and deployment**
- **30 days of post-launch support** for bugs and technical issues

### Photo Upload Upgrade Includes:
- **Photo upload functionality** on the quote request form
  - Customers can upload up to 5 photos (10MB max per photo)
  - Photos viewable in admin dashboard when viewing lead details
  - Photos stored securely via Cloudinary (free tier - 25GB storage)
- **Photo filtering** in admin dashboard (filter leads with/without photos)

### What is NOT Included:
- SMS or call automation (available as separate upgrade)
- Custom domain setup (Client responsible for domain purchase and DNS)
- Ongoing content updates or design changes beyond initial scope
- Integration with third-party tools not specified in this agreement
- Marketing, SEO, or advertising services
